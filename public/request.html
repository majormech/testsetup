Station Inventory Request

Select a station, then submit and track requests from one shared page.

No stations are available yet. Ask an admin to add one on the admin page.

How to use this page

  • Request items: Select a station, choose Request item, add one or more items and quantities, then submit.
  • Edit a request: Use Modify request on any open request to update item quantities or add/remove items.
  • Cancel a request: Use Cancel request when a request is no longer needed.
  • Track progress: Requests update live as items are issued and when the request is completed.

Status color key

  • Red/Pink: Pending request.
  • Yellow: Partially completed.
  • Green: Completed request.
  • Gray: Canceled request.
