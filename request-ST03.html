Station 3 Inventory Request

Submit requested inventory for this station.

How to use this page

  • Request items: Select Request item, add one or more items and quantities, then submit.
  • Edit a request: Use Modify request on any open request to update item quantities or add/remove items.
  • Cancel a request: Use Cancel request when a request is no longer needed (you will be asked who canceled it and why).
  • Track progress: Requests update live as items are issued and when the request is completed.

Status color key

  • Red/Pink: Pending request (not fulfilled yet).
  • Yellow: Partially completed (some items issued, some still pending).
  • Green: Completed request (all requested items fulfilled).
  • Gray: Canceled request.

Modified tag: A Modified badge means the request was edited after it was first submitted. The request card includes the modification date/time, who made the update, and the reason entered during the edit.
